Ablation study · WESAD · frozen NormWear embeddings

Does a personal physiological baseline help or hurt?

A linear probe on frozen NormWear embeddings classifies WESAD's baseline / stress / amusement protocol conditions, with an optional FiLM layer that conditions each window on that subject's own resting-state signal. This dashboard compares the FiLM and plain (no‑FiLM) arms across two generalization tests — class hold‑out and leave‑one‑subject‑out (LOSO) — at matched trainable‑parameter capacity.

Capacity matching. The plain (no‑FiLM) arm is not a smaller network — probe_model._matched_hidden_dim widens its single hidden layer so its total trainable parameter count equals a same‑hidden_dim FiLM probe's (classifier + FiLMLayer + LearnableBaselineSelector), solved in closed form to within an integer rounding residual. The ablation isolates what FiLM's conditioning buys, not model size.

Hold‑out accuracy reads low by construction. Class hold‑out trains on 80% of the two non‑held‑out classes only and tests on the remaining 20% of those plus 100% of the held‑out class — a 3‑class classifier that has never seen the held‑out class in training, evaluated on a test set that skews toward it. Precision on that class is expected to be poor; ROC AUC (which doesn't depend on the decision threshold) is the more telling column there.
